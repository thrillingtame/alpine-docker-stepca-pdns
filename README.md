Dockerize Step-CA with PDNS for use with External acme.sh Clients

Pre-Requisites:
Docker
git



Quick-Start:
git clone git@github.com:thrillingtame/alpine-docker-stepca-pdns.git
cd alpine-docker-stepca-pdns
